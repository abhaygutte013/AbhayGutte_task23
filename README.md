# Responsive Landing Page using Tailwind CSS

## Project Title

Responsive Landing Page for AppName

## Objective

The objective of this project is to create a modern and fully responsive landing page for **AppName**, a SaaS platform that helps teams build, collaborate, and launch products efficiently.

The webpage is developed using **HTML5** and **Tailwind CSS (CDN)** without writing any custom CSS. It demonstrates the use of Tailwind's responsive utilities, Flexbox, Grid system, spacing, and mobile-first breakpoints to build an adaptive layout.

## Features

- Sticky navigation bar
- Responsive navigation links
- Mobile menu icon
- Gradient hero section
- Responsive Flexbox layout
- Large heading and description
- Call-to-action buttons
- Responsive hero image
- Features section with three cards
- Hover effects on feature cards
- Responsive footer
- Mobile-first design

## How to Run

1. Download the project files.
2. Open the folder in Visual Studio Code.
3. Ensure you have an internet connection because Tailwind CSS and Font Awesome are loaded using CDNs.
4. Open `index.html` in your web browser.
